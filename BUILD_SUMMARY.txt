================================================================================
                    CUSTOMIZATION GUIDE
================================================================================

CHANGE BRAND COLORS
  Edit: src/styles/global.css
  Look for: :root { --primary: ... }
  Change the hex values to your colors

CHANGE TYPOGRAPHY
  Edit: src/styles/global.css
  Look for: --font-family, --font-sizes

CHANGE SPACING
  Edit: src/styles/global.css
  Look for: --spacing-* variables

ADD NEW PAGES
  1. Create component in src/pages/NewPage.tsx
  2. Create styles in src/styles/NewPage.css
  3. Import in App.tsx
  4. Add routing logic in App.tsx
